# Meow-Bounty
A Co-op first-person shooter game with a 3D cartoon art style.
# Abount The Game
Meow Bounty is a multiplayer co-op game where players work together to survive enemy waves, complete objectives, and reach the safe room before being overwhelmed.
# Fetures
Multiplayer co-op gameplay </br>
Lobby system </br>
Relay connection support </br>
Real-time voice chat using Vivox </br>
Various character selection </br>
Shop System </br>
Reward and progression system
Scene synchronization
# Tech Stack
## Engine
Unity
## Networking
Netcode for GameObjects (NGO)
PlayFab
## Voice Chat
Vivox
## Achitecture/Pattern
Object Pooling </br>
Strategy Pattern </br>
Service Locator </br>
Event-driven architecture
